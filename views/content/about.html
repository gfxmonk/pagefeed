Get your links delivered to your feed reader

Pagefeed allows you to add any web page to your personalised feed of pages. It's similar to (and inspired by) instapaper, except it works with whatever feed reader you want to use.

Just log in, add your (private) RSS feed to your feed reader, and you're set.

There are a few ways to add pages. Obviously, you can use the form on the main page. But you can also use the bookmarklet, which will let you add any page without visiting PageFeed. Thirdly, there is an option in GRiS (Google Reader iPhone Sync) to save links to PageFeed for viewing later.

If you want to add PageFeed integration to your own programs, get in touch and I'll see what I can do about some API documentation.

PageFeed is still in early development, so things may be a little unstable for a while. But the core functionality should be stable enough for everyday use, if you're brave and don't mind some rough edges ;)
